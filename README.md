# calculator-compose

🍉Jetpack Compose构建，基于逆波兰表达式的完整四则运算器

## 功能介绍

1. 基本的四则运算功能
2. 支持小括号运算
3. 历史记录
4. 小数计算
5. 双击 `C` 清空历史记录
